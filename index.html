Sign language educational robotic hand

Thesis

Assistive robotics prototype combining a programmable robotic hand with a custom cable-driven continuum wrist designed for sign language teaching. I designed the wrist from scratch, including calculations, CAD, 3D printing, flexibility tuning, assembly, and embedded control.

The current prototype uses camera-based recognition to detect the letter A on paper, then actuates the hand and wrist to perform the corresponding sign language gesture. The interaction layer is still under development.

XR teleoperation of Braccio robot

ROS · Unity

Developed and evaluated an XR teleoperation pipeline for a Braccio robotic arm, connecting Unity to ROS Noetic and Arduino for intuitive robot control.

The initial system used Unity, ROS–TCP Connector, URDF modeling, MoveIt, RViz, rosserial, and Arduino-based actuation. After validating the ROS-based architecture, I helped streamline the real-time loop by moving inverse kinematics into Unity to improve responsiveness and reduce latency during teleoperation.

  • Unity
  • ROS Noetic
  • MoveIt
  • URDF
  • RViz
  • Arduino

Rehabilitation exoskeleton with Brick Breaker exergame

HRI · Rehab Robotics

Developed a rehabilitation system for post-stroke upper-limb support using a 1-DOF elbow exoskeleton integrated with a Unity-based Brick Breaker exergame. The patient controls the paddle through voluntary elbow motion.

The system implements an assisted-as-needed strategy using admittance control, real-time ball trajectory prediction, and wireless UDP communication between Unity and the exoskeleton controller. The project is largely complete, with ongoing refinement of the admittance behavior.

  • Unity
  • Admittance control
  • UDP communication
  • Arduino Nano 33 IoT
  • Rehabilitation robotics
